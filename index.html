Be Human .IO

A differing ideology to how we use technology.

Download as .zip Download as .tar.gz View on GitHub

Our Primary Directive

  • Be Human: don’t act like a machine to society, acknowledge them as we did pre-telephone.
  • IO: To perform an input or output operation between human and machine; electronics are always used.

The BeHuman.IO manifesto is a guideline to be be used for projects, experiments and exportation into the unknown, all under one common goal: for the betterment of society. Be them open source, free or paid, all their underlying use is to improve and enhance ourselves and others; never to add further technical jargon and comically long terms of service we've all become accustom too.

Manifesto Framework

  1. Never act automated when speaking to another human, use first names and reframe from use of automated messages
  2. Profit can be made off another but it should bring honest happiness, never sold on F.U.D.
  3. All code used should be made open source unless otherwise bought to speed development
  4. Updates and development statuses should always be made public
  5. Never use paywalls or require email lists to gain profit
  6. Ask as little as possible personal information when billing
  7. Dumb everything down like /r/explainlikeimfive/
  8. Sharing is caring. If asked how it's done, explain
  9. Some handicaps go beyond the 5 senses, adapt for these individuals
  10. When in doubt, ask for help. And always thank for another's time
  11. N.D.A. are No Don't Agree. Always be open
  12. Two pages of terms-of-service are one page too long
  13. One mans garbage is another mans inspiration. Always show your process
  14. Embrace LGBTQA and never assume name = identified sex e.g. Mr/Ms
  15. Quit the one cent demotion listings and bump to grand total
  16. Never blame the client. Help them understand how to use your product
